Two Best Scatterplots

Plot #1

The main hypothesis: as sleep increases, stress decreases.

With this the visualization would show if there is a correlation coefficient between the two and if whether cycle phase and active minutes have an effect on stress as well.
Categorical stress, numerical active minutes, sleep in minutes and categorical cycle phases are going to be represented in this visualization.

Sleep (100 - 350 min) - x axis
Stress (Very Low, Low, Moderate, High) - y axis
Cycle Phase (Menstrual, Follicular, Ovulation, Luteal) - color of the plot points
Active Minutes (0 - 400) - size of plot dots.

How sleep affects stress across different phases of a female cycle

Scatterplot #2

The main question: how do cycle phases affect stress among different women?

This visualization would show time series graph for individual participants (ID) and would help observe how stress fluctuates across cycle phases. Parameters used in this analysis are: ID, cycle phase and stress mean.

Cycle Phase (Menstrual, Follicular, Ovulation, Luteal) - color of the time period of x axis
Stress (Very Low, Low, Moderate, High) - y axis
ID (1,2,4, et.c) - a set of points plotted on an individual graph

How stress varies across cycle phases
