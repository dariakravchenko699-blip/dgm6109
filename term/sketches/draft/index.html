Visualization #1

The main hypothesis: as sleep increases, stress decreases.

With this the visualization would show the distribution between sleep and stress categories and if whether cycle phase and active minutes have an effect on stress as well.
Categorical stress, numerical active minutes, sleep in minutes and categorical cycle phases are going to be represented in this visualization.

Sleep (100 - 350 min) - x axis
Stress (Very Low, Low, Moderate, High) - y axis
Cycle Phase (Menstrual, Follicular, Ovulation, Luteal) - color of the plot points
Active Minutes (0 - 400) - size of plot dots.

Visualization #2

The main question: how are different characteristics of well-being affected by cycle phases?

This visualization would show time series graph for individual participants (ID) and would help observe how stress fluctuates across cycle phases. Parameters used in this analysis are: day of the study, ID, cycle phase and stress.

Day in the Study (2, 4, 5 etc.) - x axis
Cycle Phase (Menstrual, Follicular, Ovulation, Luteal) - color of the time period of x axis
Stress (Very Low, Low, Moderate, High) - y axis
ID (1,2,4, et.c) - a set of points plotted on an individual graph

Visualization #3

The main hypothesis: is the same as in the second visualization.

It's a more creative representation of factors affected by cycle phases. It would be represented by IDs, days in the study, cycle phases, stress, sleep and fatigue.
The format of this particular visualization allows for more factors to be considered and visualized.

Day in the Study (2, 4, 5 etc.) - sections on the wheel
Cycle Phase (Menstrual, Follicular, Ovulation, Luteal) - colors of the wheel
Stress (Very Low, Low, Moderate, High) - green radial chart
Sleep ( 100 - 350 min) - blue radial chart
Fatigue (Very Low, Low, Moderate, High) - yellow radial chart
ID (1,2,4, et.c) - a set of points plotted for every day
