DATA COLLECTION

My dataset is a collection of tracked self-reported events and symptoms for menstrual health taken from https://www.physionet.org/content/mcphases/1.0.0/. It's an aggregated set that originally combined physical patterns such sleep and active minutes, cycle phases, stress and fatigue and different participants put down for different days in the app.

QUESTION

The main question the analysis aims to answer is if there is any relationship between cycle phases and stress. The original hypothesis: the closer to the beginning of the cycle, the more stressed the person is. The data I chose at the end (IDs, cycle phases and evrage stress) helps to see if there is any detectable negative trend between cycle sequence and stress. And to see that trend a time series scatterplot would really help.

VISUALIZATION

The plotted dots and dashed lines connecting them represent the stress trend for different participants. So far it seems liks in majority (60%) cases there is some kind of negative correlation between phases and stress. In 40%, however, there isn't a clear relatinonship to observe between them.
